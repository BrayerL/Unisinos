19:32 20/08/2019 - Complexidade de Algoritmos

Problema Computável
	-> existe um algoritmo possível que soluciona o problema
		-> máquina de touring (MT)

Complexibilidade
	-> quão fácil ou quão difícil é a solução de um problema
	-> complexibilidade de tempo
		-> ordem 
			-> constante
			-> logarítmico
			-> linear
			-> quadrático
			-> cúbico
			-> exponencial

	-> complexibilidade de espaço


Análise assintótica
	-> identifica o comportamento do algoritmo com conjuntos de dados tendendo ao infinito 
	-> apenas a tendencia de crescimento é relevante
